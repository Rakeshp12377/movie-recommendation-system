# Movie Recommendation System

This is a **Python-based movie recommendation system** using machine learning.  
It recommends movies based on similarity between movies.

## Features
- Recommend similar movies from dataset
- Uses `tmdb_5000_movies.csv` for movie data
- Machine learning model stored in `.pkl` files (not uploaded to GitHub)
- Easy to run Jupyter Notebook

## Files in this repository
- `movie-recomnder-system.ipynb` – Notebook with code
- `tmdb_5000_movies.csv` – Dataset
- `.gitignore` – Prevents large files (`.pkl`) from being uploaded
